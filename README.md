#  Race Tracker App

Android app built using Jetpack Compose and Kotlin Coroutines.

##  Features
- Two players race simulation
- Coroutine-based concurrency
- Start / Pause / Reset functionality
- Smooth progress animation

##  Tech Stack
- Kotlin
- Jetpack Compose
- Coroutines
